A Tile-based TF2 Map Prefabrication

What is the Mapper?

The mapper is a *highly efficient* program coded entirely in python. Directed at both n00bs and hammer veterans, this program eliminates the learning curve of hammer and also speeds up the map-making process for those people, respectively. Features include:

  • Creating simple maps with the press of a few buttons
  • Using prefabs to construct world geometry
  • Creating your own prefabs for use in the program

    • Now you can create your very own TF2 map without any previous knowledge of Hammer.


    You can find the mapper at http://github.com/baldengineers/easytf2_mapper. You can find instructions on the wiki for how to operate it.
