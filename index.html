A Tile-based TF2 Map Maker




What is the Mapper?

The mapper is a *highly efficient* program coded entirely in python. Directed at both amateurs and Hammer veterans, this program eliminates
the learning curve of Hammer and also speeds up the map-making process for those people, respectively. Features include:

  • Creating simple maps with the press of a few buttons
  • Using prefabs to construct world geometry
  • Creating your own prefabs for use in the program

    • Now you can create your very own TF2 map without any previous knowledge of Hammer. Currently in beta 2.6.0


    You can find the mapper at http://github.com/baldengineers/easytf2_mapper. You can find instructions on the wiki for how to operate it.
