The mapper is very simple to use, because that's what we designed it for.

    The main components of the mapper are the work area with the buttons, the list of prefabs, the rotate buttons, and the levels button. There is also a menu bar at the top of the program allowing you to perform various other functions like saving, opening, and exporting your map, creating your own prefabs, and importing community-made prefabs downloaded from the workshop. This guide is just for quick reference: for an in-depth tutorial please check out our video tutorial (currently in the making) or the tutorial on our Github page. If you are completely new to mapping, check out the TF2 Map Design Theory.

Mapper Cheat Sheet

Work Area

Contains the buttons that allow you to create geometry throughout your map.

Prefab List

Contains the prefabs to construct the maps.

Buttons used for reorganizing and removing prefabs.

Rotate Buttons

Used to rotate the prefabs in four different directions.

Levels Button

Used to change the current level of the map you are editing.

File Menu

New > Creates a new mapper project.

Open > Open a previously saved mapper project.

Save > Saves the current mapper project.

Save As > Saves the current mapper project with a new filename.

Import > Prefab > Imports a zipped prefab that was created using the program.

Export > as .VMF > Exports the current mapper project as a .VMF file, which is compatible with Valve's Hammer Map Editor.

Exit > Exits the application.

Options Menu

Change Grid Size > Changes the current grid size in the work area to whatever you set it to. You will lose any unsaved progress.

Change Skybox > Changes the current skybox to whatever you select.

Create Prefab > Used to create your own custom prefabs. Be warned: this tool has a few known bugs, so use at your own risk.

Open Hammer > Opens up hammer from whatever directory you set it to.

Developer Console > Meant purely for developers and advanced users.

Supplementary Picture Tutorial:

 Work Area

The work area contains the buttons that allow you to create geometry throughout your map. Clicking a button places the currently selected prefab in the prefab list in that spot of the grid.

 

Prefab List

The prefab list contains the prefabs to construct the maps. Click a prefab to select it. 

Prefab Control Buttons

The prefab control buttons allow you to reorganize and remove prefabs. The up arrow button moves the selected prefab up one level in the list, and the down arrow moves it down one level in the list. If you click on the trashcan icon, it prompts you to delete the selected prefab. WARNING: This action cannot be undone.

Rotate Buttons

The rotate buttons are used to rotate the prefabs in the four different directions. Click the right rotation button to rotate clockwise, and the left rotation button to rotate counter-clockwise. The current rotation is shown in the middle of the two buttons

Levels Button

The levels button is used to change the current z level of the map you are editing. Click on the button to open a menu that you can use to change the current level you are on. These "levels" are like the stories of a building: level 1 is the ground level, and as you move up in levels, the tiles' distance from the ground increases. The arrows located next to the button are a shortcut to move up/down a level

Tools Menu

Create Prefab

Used to create your own custom prefabs. Be warned: this tool has a few known bugs, so use at your own risk.

  1. Prefab Text > Enter the name of the prefab as it will appear in the prefab list on the right.
  2. Prefab Name > Enter the filename of the prefab. No spaces allowed.
  3. VMF File > The vmf file for the prefab (exported from hammer)
  4. Icon > The jpg file containing the icon as it will appear in the prefab list on the right.
  5. Make Rotations > Check this if you would like to create rotated version of the prefab.
  6. Export Prefab > Check this if you would like to also save the prefab as a *.zip to upload to the workshop.

Open Hammer

Opens up hammer from whatever directory you set it to.

Developer Console

Meant purely for developers and advanced users. (Try typing "pootis")
